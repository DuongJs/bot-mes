HƯỚNG DẪN SỬ DỤNG BOT
===================

1. Cấu hình:
   - Mở file 'config.json' bằng Notepad hoặc trình soạn thảo văn bản.
   - Nhập cookie Facebook của bạn vào các trường tương ứng:
     - "c_user": ID người dùng của bạn.
     - "xs": Cookie xs.
     - "fr": Cookie fr.
     - "datr": Cookie datr.
   - Bạn có thể lấy các cookie này từ trình duyệt (F12 -> Application -> Cookies).
   - Lưu file lại.

2. Chạy Bot:
   - Nhấp đúp vào file 'bot.exe'.
   - Bot sẽ khởi động và cửa sổ đen (terminal) sẽ hiện ra. Đừng tắt cửa sổ này.

3. Quản lý:
   - Mở trình duyệt và truy cập: http://localhost:8080
   - Tại đây bạn có thể xem trạng thái bot và cập nhật cấu hình.

4. Sử dụng:
   - Chat với bot trên Messenger hoặc trong nhóm mà bot tham gia.
   - Gõ '!ping' để kiểm tra.
   - Gửi link Facebook/TikTok/Instagram để bot tải video và gửi lại.
